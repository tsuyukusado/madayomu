# Gitを活用しよう
『Gitっていうのは、いわばセーブポイントを作れる魔法なんだ』
「セーブポイント……それが、｜引き返せる《圏》ってこと？」
『あぁ、そういうことだ』
　Gitの導入方法については、ネット検索やそれこそAIに聞けば教えてくれると思う。基本的にターミナルでの操作だから、エラーが出てもAIに丸投げすれば対処法は分かるはずだ。
『例えば今のコードは、`Hello Madayomu!`って文章が出るだけだ。でもこの先、色々な機能をどんどん追加していくことになる』
「うん」
『そんな時、こまめにセーブしておけば、だ。「この機能だけ無くしたい！」って時はその時のセーブポイントに戻ってその変更だけを取り消す――Gitがあれば、そんなことも可能になる』
「その変更だけって……全部巻き戻らずに、｜それだけを《圏》ってこと？」
『あぁ、そうだ。試してみるか？』
「うん」
　アイはパソコンに潜り込むと、`test.md`というファイルを生成した。
『まず、適当な文章を書いてみる』

```md
# これはテスト文章だぜ
- Gitの力、見せつけてやるよ！
```

『最初はまず、データを保存する。そうしたらGitの画面に行って、**変更をステージ**をクリックだ。この、プラスマークだな』

『よし、いいぜ。そうしたらあとは、そのまま下のボタンをクリックだ！』
「うん！」
　すると下のグラフというところに、僕が入力した文言が表示された。
『今のが最初のコミット。次に、少しだけデータを加筆してみよう』

```md
# これはテスト文章だぜ
- Gitの力、見せつけてやるよ！
- 一体どうなるんだろう？
```

「こんな感じ？」
『あぁ。それをもう一度コミットだ』
「うん」
　今度は`ドキドキ…ワクワク…`という名前にしてコミットすると、アイが『なんだそれ』とくすくす笑う。
「もう。僕が分かれば良いんだろう？」
『ふふ。まぁそうだけどな」
　２つ目のコミットが終わると、アイが『早速だが、ここをクリックしてみてくれ』と画面を指さした。
「どれ？」
『下のグラフってところに書かれてる、`ドキドキ…`の部分だ。クリックすると、下に変更したファイルの一覧が出てくる』
「うん…本当だ、出たね」
『その`test.md`をクリックしてみろ。右に出てくる画面に注目だ』
「うん……あ！」

「なんか緑になってるね！」
『これが**追加した行**だ。ちなみに、消去した行は赤く表示されるぜ』
「なるほど……これだけでも便利だね」
『Gitの便利なところはこれだけじゃないぜ。早速セーブポイントへの巻き戻しを試してみよう。`リバート`って機能だ』
「う、うん」

　アイはターミナルに、以下のコマンドを打ち込んだ。

```zsh
git revert dd26140aa4e679eeedf35adada48d4358ad3a42f
```

「こ、この英数字の羅列はなに!？」
『これはコミットIDだ。`ドキドキ…`の部分を右クリックすると取得出来る、コミット一つ一つに割り振られた番号だぜ。ちなみに、最初の7文字だけでも動作はするな』
「な、なるほど……それで、これを実行する……」
　僕がエンターボタンを押すと、ターミナルで何かが走った。
『よし。コミットメッセージを見てみろ。`Revert ...`って書かれてるだろ』
「うん。これをコミットすればいいんだね」
『あぁ、そうだ』
　試しに僕はコミットボタンを押した。これだけで、本当に巻き戻るのかな？　と半信半疑になりながら。すると――。

「消えてる！　消えてるよアイ！」
『ほら、言っただろ？』
　アイは、ふふーんと自慢気に胸を張る。僕はこくこくと頷きながら、VScodeの画面を食い入るように見ていた。

　これが、Gitの力…！　これがあれば、試行錯誤も安心してできる！

『他にも、｜セーブポイントを枝分かれさせる《圏》**ブランチ**や、そのブランチをくっつける**マージ**なんかもある。さらに、そのセーブポイントをネット上に公開する機能――**GitHub**もあるんだぜ』
「も、も、盛り沢山だね」
『それらの詳細は割愛するが、使いこなしていけば自由自在にコードが書けるってわけだな』
　夢のような機能だ。この知識を踏まえていれば、一歩ずつ、着実に歩みを進められるに違いない。
　僕は再びメモでGitの使い方を復讐した。
　その後、改めてアイに向き直る。
「それじゃあ、今度こそ……マダヨムの実装に進む？」
　僕の問いかけに顔を上げたアイは、脚を組んでにんまりと笑ってみせた。

『あぁ……そうしよう！』
