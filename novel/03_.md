# コーディングを始めよう
## 僕の"hello world"

『よし、ここまで来たら後は作るだけだ』

　行くぞ、と拳を握って訴えてくるアイを見下ろして、僕は冷や汗を流した。

「本当に出来るのかな……」
『出来るさ。お前には俺が居るんだぜ？　何をしたら良いのか分からなかったら、そのままそう俺に尋ねればいいのさ』
「えっと……それなら……」

　僕はおずおずと、アイに向かって問いかける。

「僕は何から始めればいい？」
『よし来た』

　満足気に笑ったアイは、再びパソコンの中へと潜り込んでいく。

『まずはフォルダを整理しよう。今はこの根っこのフォルダに、そのまま２つの書類が入っている。`docs`(書類)」っていうフォルダを作ってこの中にまとめよう』
「わ、わかった」
『そんでもって、次は`src`ってフォルダを作れ』
「src？」

　僕が首を傾げると、アイは『ああ』と頷いた。

『`src`はな、お前が書くコードを入れるフォルダだ。`source code`の略。ソースコードっていうのは、コンピュータと人間をつなぐ境目だと思ってくれりゃいい』
「コンピュータと人間を、つなぐ……」

　僕が反芻すると、アイはもう一度頷く。

『まぁつまり、プログラミングって言われてイメージするような英数字の羅列――**お前が自分の意志で書いたコード**は、全部ここに入ると思ってくれればいい。そのためのフォルダだ』
「なるほど……」

　言われるがままに、僕はフォルダを作った。

『そしたら、実際に`src`の中にファイルを作っていこう。そうだな……今回お前は、最終的にどうやってマダヨムを公開したいんだ？』
「スマホでもパソコンでも、誰でも気軽に使えたら良いなって思っているよ」
『それなら言語はDartにしようか』
「だーと？」
『Dartって言うのは、マルチプラットフォームに対応した言語なんだ』
「マルチプラットフォーム…」

　マルチプラットフォームというのは、PC,iOS,Android,などの色々なプラットフォームに対応できるということ。それが叶えば、誰でも自由にPDFが生成できる世界が実現する――そう思う。

「それがいい！　勉強したことないけど、Dart、試してみたい！」
『了解だ。それなら俺が、１からレクチャーしてやるよ』

　まずは心臓部を作るぞ。そう言って、アイはこんこんとPCの画面を叩いた。

『今作った`src`フォルダに、`main.dart`を作るぞ』
「えっと…僕が作ったらいいの？」
『なんなら、そこから俺の力を借りたっていいんだぜ？』
「いいの？」

　アイは画面の中に再び潜り込むと、くるくると指を回した。
　すると、新しいファイルが生成される。

```dart
// main.dart

void main() {
  print('Hello World');
}
```

『ほらな』
「出来た……！」

　アイが魔法のように書いてくれたコードを見て、目が輝く。若しかしてこれは、実行したら`Hello World`と出るのだろうか。

「アイ、アイ、実行してみたい！」
『よし来た。コマンドは`dart run src/main.dart`だぜ』
「わかった！」

　アイがテキストでコマンドを表示してくれたので、僕はそれをターミナルに張り付けた。
　ところが……。

「あれ？　エラーが出たよ？」
『ん？　……あぁそうか、お前まだDartを入れてなかったな』
「入れないといけないんだ」
『おう、そうだな。手順を順に説明していくぞ』

　僕はアイの話を聞きながら、順に作業を進めていった。
　この間、自分でどこかを検索したりは一切していない。アイに言われるがままだ。
　手順をまとめると、以下の通りになる。ちなみに、以下はWindowsでの導入方法だ。他のOSに関しても、アイに聞けば教えてくれると思う。

1. https://gekorm.com/dart-windows/ にアクセスしてインストール。
2. VScodeを開きなおす。

「これだけでいいの？」
『あぁ。どうやら現行の仕様だと、環境変数が勝手に設定されるみたいだから問題ないよ。試しにターミナルで`dart --version`うを叩いてみろ』
「う、うん」
　言われるがまま、試してみると……。

「バージョンが出た！」
『やったな！　それじゃあ改めて、打ってみようぜ、さっきのコードをさ』
「うん！」

　もう一度チャレンジした結果は……。

```zsh
Hello World
```

「出たー！」
『やったな！』

　なんだろう、この達成感。
　ハローワールドという文字を出しただけなのに。それなのに、何かが動いたという事実が僕に喜びを与えてくれる。
　胸がわくわくと膨らんでいくような感覚が止まらない。僕はアイの方を見て、思わず立ちあがった。

「このままどんどん行こう！　次は何をすればいい!?」
『ははは、まぁ落ち着けよ。まずはこのコードへの理解を深めようぜ』
「理解？」
『この先俺と一緒にコーディングをすることになるわけだが、俺だって完璧じゃない。バイブコーディングの基本は、俺が書いて、お前がチェックするっていう構造だ。だからコードを少しずつ書いていきながら、お前自身も少しずつ学んでいく必要があるんだよ』
「そうか……」

　難しいのだろうか。思わず俯いてしまう僕を見て、画面から戻ってきたアイがぽんぽん手の甲を叩いてくる。

『大丈夫だ。基礎をおさえてさえいれば、中の詳細がブラックボックスだろうと問題ないと俺は思うぜ』
「そうなの？」
『おう。ひとまず今のコードを解説するな』

　これが最初のステップだ。そう言って、アイは一文ずつカーソルを動かして説明してくれた。

```dart
void main(){コード}
```

『まずはこれだ。これが、すべての入り口。コードが動くと、まずここを探しに来る』
「ふむふむ……全部をこの中に入れるってこと？」
『そんな感じだな』

　次に、これだ。アイが示したのは、その次のコードだった。

```dart
print('Hello World');
```

『これは英単語から、なんとなくわかるんじゃないか？』
「うん。プリント…印字ってことだよね。ターミナルに出力するやつだ」
『そうそう』

『ほら、どうだ？　こうしてひとつずつ追いかけていくと、理解できるだろ？』
「うん」

　一気に進めていくと呪文の羅列のように思えてしまうコードが、こうして読み解くことで理解できる文章になる。それが僕は、どことなく嬉しかった。

「ふふ……なんだか楽しい。少しずつ、形になっていく喜びって言うのかな」

　思わずそう零した僕を、アイは満足気に見上げた。

『喜ぶのはまだ早いぜ。PDFすらできてないじゃないか』
「それはそうだけど……って、まさかもう作れるの？」
『俺を誰だと思ってるんだ？』

　アイは、再びパソコンの中へ潜っていくと、サラサラコードを書き始めた。

```dart
import 'dart:io';
import 'package:pdf/widgets.dart' as pw;

void main() async {
  final pdf = pw.Document();

  pdf.addPage(
    pw.Page(
      build: (pw.Context context) => pw.Center(
        child: pw.Text('Hello Madayomu!'),
      ),
    ),
  );

  final file = File('example.pdf');
  await file.writeAsBytes(await pdf.save());
  
  print('PDFができたぜ、ユージオ！');
}
```

『よし、そして…』
