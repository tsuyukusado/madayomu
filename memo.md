# 各種ファイルの役割
src/main.dart: アプリケーションのメイン処理（ファイル読み込み、PDF生成フロー）
src/models.dart: データモデル（禁則処理用アイテム、目次エントリーなど）
src/utils.dart: ユーティリティ関数（テキスト解析、シンタックスハイライト）
src/widgets.dart: カスタムウィジェット（ページ番号記録用ウィジェットなど）
pubspec.yaml: ライブラリの読み込み

# テスト項目
- 表紙が正常に表示されているか
- ページ番号が適切か
- 「もう、寝た方が」の行の行頭スペース
- 行頭スペース
- 改行間隔が均一か
- 幻想のあとの改行

# 用語解説
- **データモデル**: プログラムで扱うデータの「型」や「構造」を定義したクラスのこと。処理ロジックを持たず、データを保持する箱のような役割を果たします（例：`TocEntry`は目次の1行分のデータを表す）。
- **エントリー**: 「項目」や「登録データ」のこと。リストや目次などの一覧における、一つ一つの要素を指します。
- **ウィジェット**: 画面やPDF上に表示される「部品」のこと（テキスト、画像、レイアウト枠など）。
- **カスタムウィジェット**: 標準で用意されているウィジェットを組み合わせて作った、独自の機能を持つ部品のこと（例：`PageRecorder`）。
